# 来点音乐

* 写代码累了吗 来点音乐吧
* 基于vue.js@2.6.9
* 数据均来自qq音乐 通过jsonp获取

## 安装所需要的包
```
yarn install
```

### 启动服务 本地浏览器打开localhost:8080
```
yarn run serve
```

### 打包项目
```
yarn run build
```
